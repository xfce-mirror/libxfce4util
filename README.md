libxfce4util
====================

libxfce4util is used to share commonly used non-GTK+ utilities among the Xfce applications.

### Installation

From source: 

    % cd libxfce4util 
    % ./autogen.sh
    % make
    % make install

From release tarball:

    % tar xf libxfce4util-<version>.tar.bz2
    % cd libxfce4util-<version>
    % ./configure
    % make
    % make install
